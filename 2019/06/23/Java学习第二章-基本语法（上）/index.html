Java学习第二章 基本语法（上）

Author: Wysam Date: 六月 23, 2019  14:27:04 Category: Java学习

注释

类型 符号 备注
单行注释 // 注释只在源码有，在字节码文件中没有，故编译程序编译时会跳过注释语句
多行注释 /**/ 不能嵌套使用
文档注释 /** */ 包含一些说明性文字及一些JavaDoc卷标，后期写项目时，可生成项目的API

标识符

基本规则

  • 以字母、下划线_、美元符号$开头
  • 其它部分除了上述三种外，还可以是数字
  • 大小写敏感，长度无限制
  • 不可以是 Java 的关键词

使用规范

类型 规范
类名 每个单词首字母大写，如 Man, GoodMan
方法、变量 第一个单词小写，第二个开始首字母大写，如 eatFood()，采用驼峰原则
常量 全大写，单词用_ 隔开，如 AGE_MAX

变量

名称 定义 生命周期 备注
局部变量
local variable		 方法或语句块内部定义的变量 从声明位置开始到方法或语句块执行完毕为止 必须初始化
成员变量
（实例变量）
member variable		 方法外部、类的内部定义的变量，不使用static关键词 从属于对象，伴随对象始终 自行初始化
静态变量
（类变量）
static variable		 使用static定义 从属于类，从类加载到卸除

成员变量初始化值

基础数据类型 值
int 0
double 0.0
char ‘\u0000’
boolean false

常量

类型 举例
一般常量 1、‘a’、true、“你好”
符号常量 final String NAME=”张三”

数据类型

基本数据类型

整型

类型 占用存储空间(byte) 位数(bit)$^{1byte=8bit}$ 表数范围
byte 1字节 $2^8(256)$ $-2^7$ ~$2^7-1$(-128~127，负号和0分别占1位)
short 2字节 $2^{16}(65536)$ $-2^{15}$ ~ $2^{15}-1$(-32768 ~ 32767)
int 4字节 $2^{32}$ $-2^{31}$ ~ $2^{31}-1$(约正负21亿)
long 8字节 $2^{64}$ $-2^{63}$~ $2^{63}-1$

  • 整型常量默认是 int 类型，当超过21亿时，要在末尾加 L(l)，转为 long 类型

  • 整型常量的表现形式

    类型 说明
    十进制 如99，-500
    八进制整数 以0开头，如015
    十六进制数 以0x或0X开头，如0x15(0~9,a~f)
    二进制数 以0b或0B开头，如0b011100

浮点型

类型 占用存储空间 表数范围 备注 强制转换
float
单精度类型		 4字节 -3.403E38 ~ 3.403E38 尾数精确到7位有效数字
赋值时须加后缀，否则损失精度		 末尾加F(f)
double
双精度类型		 8字节 -1.798E308 ~ 1.798E308 浮点型数据的默认类型 末尾加D(d)

  • 科学计数法

    $$314e2=314*10^2=31400$$

    $$314E-2=314*10^{-2}=3.14$$

  • 浮点数是不精确的，不能直接比较

    • 情况一：0.1f==1.0/10d，结果为 false
    • 情况二：422356589f==422356589f+1，结果为 true

  • 调用 java.math 包下的两个类来比较

    • BigInteger：实现任意精度的整数运算

    • BigDecimal：实现任意精度的浮点运算

字符型

  • char：占2字节

  • Java采用 Unicoode 标准国际字符集，包含了汉字，拥有从0到65535之间的编码，通常用 '/u0000' 到 '/uFFFF' 之间的十六进制值来表示

  • 转义符

    转义符 含义 Unicode值
    \b 退格（backspace） \u0008
    \n 换行 \u000a
    \r 回车 \u000d
    \t 制表符（tab） \u0009
    \‘’ 双引号 \u0022
    \‘ 单引号 \u0027
    \\ 反斜杠（\） \u005c

布尔型

  • 有两个常量值：true/false ，只占1位，所以不能用0或非0的整数来替代
  • 最好先声明 boolean 变量并赋值，再放入控制语句中，以写错
