Spring

Author: Wysam Date: 七月 10, 2019  20:53:34 Category: Java框架

简介

Spring是分层的Java SE/EE应用full-stack轻量级开源框架，以IoC（Invdrse Of Control：反转控制）和AOP（Aspect Oriented Programming：面向切面编程）为内核，提供了展现层Spring MWC和持久层Spring JDBC以及业务层事务管理等众多的企业级应用技术，还能整合开源世界众多著名的第三方框架和类库，逐渐成为使用最多的JavaEE企业应用开源框架

IOC/DI(控制反转/依赖注入)

准备核心配置文件applicationContext.xml

在测试类通过spring获取Category对象，以及该对象被注入的name属性

Product类中有对Category对象的setter getter

在配置文件中用ref注入另一个对象（Category）

于是在测试类中product1可以调用category1的方法和属性

注解方式

在beans的标签的首行添加，告诉Spring要用注解的方式进行配置。并去掉之前的注入

  • @Autowired

    • 在Product类中有关category的属性前、构造器前，或Setter前，加上@Autowired

    • 按属性的类型查找，如有多个属性名，则报错

  • @Resource

    • 同样的位置，可以在参数中指定Category对象的关键字。

    • 这样即使配置文件中有多个Category对象，也能顺利找到

  • @Component

    • 移除配置文件中的所有信息，并新增

      告诉Spring，bean都放在这个包下

    • 在两个类前加上@Component注解，即表明此类是bean

    • 将属性初始化放在属性声明上

    • 需搭配@Autowired使用

AOP(面向切面编程)

功能分类

  • 核心业务功能：登陆，增加数据，删除数据等

  • 周边功能（切面）：性能统计，日志，事务管理等等

  • 在面向切面编程的思想里面，核心业务功能和切面功能分别独立进行开发。然后把切面功能和核心业务功能 “编织” 在一起，在登陆的时候，可以看到日志输出了

操作过程

  • 业务类

  • 切面类

    在配置文件中指定around后，执行业务的前后可以自定义辅助操作，甚至可以改变传入的参数

    Object object = (Object) joinPoint.proceed(new Object[]{“abc”});

  • 配置文件

    1. pointcut：切点。相当于一个触发器，一旦表达式中的方法执行了，即触发切面

      expression=” execution(* test2.ProductService.*(..))”

      • execution：连接点
      • 第一个*：返回任意类型，包括void
      • test2.ProductService.：表示test2包下ProductService类的任意方法
      • (..)：表示方法的参数是任意数量和类型

    2. 声明切点的另两种方式

      • 将切点Bean声明在切面中：一般该切点只被该切面使用

      • 匿名切入点Bean：该切点只被该通知使用

    3. aspect：切面

      • ref：表示引用的类，即这个切面执行的是id为”LoggerAspect”的类中的方法
      • around：表示包裹了被通知的方法，在被通知的方法调用之前和调用之后执行自定义的行为
      • pointcut-ref：指定切点，建立了切点与切面的连接
      • method：通知执行时，调用的代理类（切面）中的方法

    4. 除 around 外，其它四种通知形式

      before——在方法调用之前调用通知

      after——在方法完成之后调用通知，无论方法执行成功与否

      after-returning——在方法执行成功之后调用通知

      after-throwing——在方法抛出异常后进行通知

  • 用注解执行AOP

    • 用@Component(“s”) 注解业务类，表明这是一个bean，交由Spring管理

    • 用@Aspect注解切面类，@Component表明这是一个bean，@Around(value = “execution(* com.how2java.service.ProductService.(..))”)表明在ProductService类中的所有方法执行的*前后**进行切面操作
